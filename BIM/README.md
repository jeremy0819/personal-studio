# BIM 學習資產（Revit ＋ Navisworks，對接 BIM 職缺）

這是一套**以實際 BIM 職缺反推**的 Revit 實戰學習教材，給「建築系畢業、會基本建模、想轉進 BIM 職位」的人。

## 這包東西怎麼用

1. **每次開新對話，先讀 `CLAUDE.md`** —— 它是跨 session 記憶，會告訴 AI 顧問：我是誰、目標、目前進度。
2. **先看 `docs/`** —— 三份奠基心法（核心思維、爭議、自我鑑別題），是整門課的地基。
3. **照 `curriculum/00-課程地圖.md` 走** —— 從 M1 開始，每個模組都對應職缺的某一條職責。
4. **用 `progress/進度追蹤.md` 記錄** —— 每完成一課勾一格，並累積「可寫進履歷」的句子。

## 資料夾結構

```
BIM/
├─ README.md              ← 你在這裡
├─ CLAUDE.md              ← 跨 session 記憶（最重要，先讀這個）
├─ docs/                  ← 奠基心法
│  ├─ 00-核心思維.md
│  ├─ 01-三大爭議.md
│  └─ 02-鑑別問題.md
├─ curriculum/            ← 課程
│  ├─ 00-課程地圖.md       ← 全局路線（M1→M6）
│  └─ M1-模型建置與深化.md  ← 目前進行中
└─ progress/
   └─ 進度追蹤.md          ← 勾選清單 + 履歷句型
```

> 這包資料夾設計成可以**整包複製到任何新 repo 的根目錄**直接用（比照 `personal-studio` 的做法）。
