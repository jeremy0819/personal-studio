## 1. 敷地計劃是什麼？

### 1.1 定義

**敷地計劃（Site Plan）** 是建築申請送照時，說明基地如何配置建物、停車、景觀、法定空地、退縮等空間關係的計劃書與圖說。

> 白話說：**「這塊地打算怎麼用？建物放哪？剩下的地怎麼處理？」** 的完整書面說明。

### 1.2 與建照申請的關係

```
建照申請送件流程
│
├── 敷地計劃書（含計算書）  ← 本教程重點
├── 建築設計圖（平立剖面）
├── 結構計算書
├── 消防計劃書
└── 其他專項計劃
```

### 1.3 開發業者為什麼要懂敷地計劃？

| 場景 | 如果不懂的後果 |
|------|--------------|
| 前期評估 | 容積試算錯誤 → 銷坪高估 → 財務失準 |
| 委外設計 | 看不懂建築師圖說 → 無法確認法規合規 |
| 都更協議 | 不知道可蓋多高、退縮多少 → 談判缺乏依據 |
| 建照審查 | 補件來回 → 工期延誤 → 資金成本增加 |

## 2. 核心法規架構

### 2.1 主要依據法規（台灣）

| 法規 | 主要管制內容 |
|------|------------|
| **建築法** | 建築申請程序、使用執照 |
| **建築技術規則（建築設計施工編）** | 退縮、容積、建蔽率、停車、法定空地 |
| **都市計劃法** | 土地使用分區、容積率上限 |
| **都市更新條例** | 都更範圍內容積獎勵 |
| **危老重建條例** | 危老容積獎勵、補助機制 |
| **各縣市土地使用分區管制自治條例** | 地方附加管制（如台北市、新北市各有差異） |

### 2.2 分區與基準容積對照（常見住宅區）

| 使用分區 | 建蔽率上限 | 容積率上限（基準值，依各縣市） |
|----------|-----------|----------------------------|
| 第一種住宅區（住一） | 30% | 60% |
| 第二種住宅區（住二） | 35% | 120% |
| 第三種住宅區（住三） | 45% | 225% |
| 第四種住宅區（住四） | 50% | 300% |
| 商業區 | 70% | 560%（依縣市差異大） |

> ⚠️ 以上為通例，實際數值以**各縣市都計書及分區管制規則**為準，每案務必查詢地籍謄本標示之分區。

### 2.3 容積獎勵疊加邏輯（都更／危老）

```
可建容積 = 基準容積 × (1 + 獎勵比率總和)

獎勵項目範例（危老）：
  ├── 結構安全性能提升：最高 30%
  ├── 綠建築標章：最高 10%
  ├── 智慧建築標章：最高 10%
  ├── 基地面積獎勵：基地 ≥ 1,000㎡ 最高 10%
  └── 時程獎勵（容積20%）：申請期限內
```

## 3. 敷地計劃書的必備內容

### 3.1 書面計劃書（文字部分）

1. **基地概述**
   - 基地位置（縣市、地段、地號）
   - 使用分區及特別管制事項
   - 基地面積（公告面積 vs 實測面積）
   - 道路寬度（臨路面數與寬度）

2. **法規查核摘要**
   - 建蔽率計算
   - 容積率計算（含獎勵）
   - 退縮規定
   - 停車位計算
   - 無障礙設施規劃

3. **配置說明**
   - 建物棟數、樓層數
   - 各用途樓地板面積彙整
   - 法定空地位置與面積
   - 開放空間說明（若適用）

4. **環境影響評估（若需要）**
   - 日照分析
   - 通風分析
   - 景觀敏感區說明

### 3.2 必備圖面

| 圖號 | 圖面名稱 | 比例 | 重點標示 |
|------|---------|------|---------|
| S-01 | 位置示意圖 | 1/1200 或 1/600 | 基地四至、周邊道路 |
| S-02 | 地籍套圖 | 1/500 | 地號、道路寬度、地界 |
| S-03 | 配置圖 | 1/200 或 1/300 | 退縮線、建物外廓、停車 |
| S-04 | 法定空地圖 | 1/200 | 法定空地範圍、面積計算 |
| S-05 | 停車平面圖 | 1/100 | 車位編號、尺寸、迴車動線 |

## 4. 關鍵計算公式與查核要點

### 4.1 建蔽率計算

```
建蔽率 = 建築面積 ÷ 基地面積 × 100%

建築面積 = 各層最大投影面積（不含免計項目）

免計入建蔽率項目（常見）：
  - 地下層（含地下停車場）
  - 陽台（符合規定尺寸內）
  - 法定騎樓
```

**查核重點：**
- 建築面積是否超過上限
- 屋簷挑出超過 1.5m 以上部分是否已計入

### 4.2 容積率計算

```
容積率 = 各層樓地板面積總和 ÷ 基地面積 × 100%

樓地板面積總和 = Σ（各層實際樓地板面積）
              - 免計容積項目

常見免計容積項目：
  - 停車空間（附建停車位）
  - 機電設備空間（限定比率）
  - 屋頂突出物（電梯機房、樓梯間）
  - 開放空間獎勵（若適用）
```

**查核重點：**
- 免計項目是否有超過法定比率上限
- 地下一層部分用途是否需計入容積

### 4.3 退縮計算

```
退縮建築線 = 道路中心線 + （指定道路寬度 ÷ 2）+ 附加退縮

附加退縮規定（台北市住宅區範例）：
  - 建築物高度 ≤ 15m：退縮 2m
  - 建築物高度 15m < H ≤ 21m：退縮 3m
  - 建築物高度 > 21m：退縮 4m
```

> ⚠️ 各縣市退縮規定差異極大，務必查詢當地法規。台北市另有「建築基地綠化自治條例」要求。

### 4.4 停車位計算

```
應設停車位數 = 住宅樓地板面積 ÷ 每輛所需面積標準

台北市範例（住宅）：
  - 每 75㎡ 樓地板面積設 1 輛（基本）
  - 機械停車位換算：1 機械位 = 0.7 輛（依規）

停車空間面積（每輛）：
  - 小型車：長 5.5m × 寬 2.5m
  - 機械停車：依廠商規格另計
```

## 5. 都更／危老特別注意事項

### 5.1 危老重建敷地計劃重點

**基地條件確認：**
```
危老申請資格查核清單
□ 建物屋齡 ≥ 30 年（民國○年以前完工）
□ 結構安全評估（耐震能力不足 or 海砂屋/輻射屋）
□ 基地面積 ≥ 200㎡（或符合例外規定）
□ 全體所有權人同意（或 10/10）
```

**容積獎勵計劃書附件：**
- 結構安全性能評估報告（取得獎勵必備）
- 綠建築設計說明（若申請綠建築獎勵）
- 智慧建築設計說明（若申請智慧建築獎勵）

### 5.2 都市更新敷地計劃重點

**事業概要 vs 事業計劃：**
```
事業概要階段（較早）：
  └── 概略敷地計劃即可（位置、規模、容積試算）

事業計劃階段（送審）：
  └── 完整敷地計劃書（含詳細計算書、圖面）
```

**更新後敷地計劃書特殊內容：**
- 更新前後對照表（現況 vs 計劃）
- 公共設施用地比例說明
- 原地主分回比例試算依據
- 開放空間留設計劃

### 5.3 整合型基地的敷地計劃挑戰

| 問題 | 說明 | 對策 |
|------|------|------|
| 地號整合 | 多筆地號合併計算容積 | 先辦地籍整理或合件申請 |
| 分區不同 | 基地跨不同使用分區 | 各分區容積分開計算後加總 |
| 道路退縮不一致 | 不同側道路寬度不同 | 逐條邊界分別計算退縮 |
| 既有建物保留 | 都更不拆部分建物 | 分棟計算容積，再合併確認 |

## 6. 常見審查缺失與對策

### 6.1 建蔽率 / 容積率類

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 建蔽率超過上限 | 陽台外突計算有誤 | 重新量算各層最大投影面積 |
| 容積率超過上限 | 免計項目誤計 | 逐項確認免計依據法條 |
| 計算基礎有誤 | 用公告地積而非實測地積 | 以地政事務所實測成果為準 |

### 6.2 退縮 / 配置類

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 退縮深度不足 | 漏算道路拓寬計劃 | 查詢都計書預定道路計劃 |
| 建物跨越建築線 | AutoCAD 圖面精度問題 | 以坐標套算，精度至 cm |
| 停車迴車空間不足 | 機械停車規劃過密 | 依機械停車廠商圖面配置 |

### 6.3 法定空地類

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 法定空地不足 | 建蔽率算到上限時空地剛好不足 | 反推：先計算法定空地最小面積再配置建物 |
| 法定空地被覆蓋 | 地下室頂板未計入透水面積 | 確認綠化及透水規定，分開計算 |

## 7. 自我檢查清單（送件前）

### 7.1 基本資料確認

```
□ 地籍謄本（近3個月）
□ 地籍圖謄本（確認地界）
□ 都計書（確認使用分區、道路計劃）
□ 建管單位指定建築線（書面或圖面）
□ 道路現況照片（四至方向各一張）
```

### 7.2 計算書查核

```
□ 建蔽率計算書（附圖面標示）
□ 容積率計算書（逐層列表）
□ 各層免計項目計算（有法條依據）
□ 停車位計算書（住宅、商業分開）
□ 無障礙設施數量確認
□ 法定空地面積計算書
```

### 7.3 圖面一致性

```
□ 配置圖退縮線標示正確
□ 各圖面比例標示一致
□ 北向箭頭方向正確
□ 樓層標示（FL、GL）定義一致
□ 建物外廓尺寸與計算書一致
```

### 7.4 都更／危老特別查核

```
□ 容積獎勵計算有文件依據
□ 獎勵申請書類與計劃書數字一致
□ 原建物屋齡證明文件備齊
□ 都更事業計劃（或危老申請書）與敷地計劃一致
```

## 8. 名詞解釋速查

| 名詞 | 英文 | 說明 |
|------|------|------|
| 敷地計劃 | Site Plan | 基地整體配置計劃書 |
| 基地面積 | Site Area | 計算建蔽率、容積率的基準面積 |
| 建蔽率 | Building Coverage Ratio (BCR) | 建築面積÷基地面積 |
| 容積率 | Floor Area Ratio (FAR) | 樓地板面積總和÷基地面積 |
| 建築面積 | Building Footprint | 建物各層最大水平投影面積 |
| 法定空地 | Required Open Space | 建蔽率管制下的未建築地 |
| 退縮 | Setback | 建物退離道路境界線的距離 |
| 建築線 | Building Line | 建管指定的建築許可起造線 |
| 指定建築線 | Designated Building Line | 建管單位依道路計劃指定的線 |
| 容積獎勵 | FAR Bonus / FAR Incentive | 符合條件可額外增加的容積 |
| 基準容積 | Base FAR | 土地使用分區管制的容積上限 |
| 可建容積 | Allowable FAR | 含獎勵後實際可使用的容積 |
| 樓地板面積 | Floor Area | 各層樓板面積（含牆體） |
| 免計容積 | FAR Exemption | 不納入容積計算的面積項目 |
| 附建停車位 | Required Parking | 法定應附設的停車空間 |
| 開放空間 | Open Space | 供公眾使用的留設空地 |
| 無障礙設施 | Barrier-free Facilities | 供行動不便者使用的設備設施 |
