## 1. 敷地計劃是什麼？

### 1.1 定義

**敷地計劃（Site Plan）** 是建築申請送照時，說明基地如何配置建物、停車、景觀、法定空地、退縮等空間關係的計劃書與圖說。

> 白話說：**「這塊地打算怎麼用？建物放哪？剩下的地怎麼處理？」** 的完整書面說明。

### 1.2 與建照申請的關係

```
建照申請送件流程
│
├── 敷地計劃書（含計算書）  ← 本教程重點
├── 建築設計圖（平立剖面）
├── 結構計算書
├── 消防計劃書
└── 其他專項計劃
```

### 1.3 開發業者為什麼要懂敷地計劃？

| 場景 | 如果不懂的後果 |
|------|--------------|
| 前期評估 | 容積上限搞錯 → 規模估算整個偏差 |
| 委外設計 | 看不懂建築師圖說 → 無法確認法規合規 |
| 都更協議 | 不知道可蓋多高、退縮多少 → 談判缺乏依據 |
| 建照審查 | 補件來回 → 工期延誤 → 資金成本增加 |

## 2. 都市計畫層級架構（最重要的基礎）

> 這是本教程最核心的概念。敷地計劃所有的數字（容積率、建蔽率、退縮）**都不是憑空而來的**，
> 而是從上面幾個層級的計畫「一層一層往下訂出來的」。
> 搞清楚層級，才知道要去哪裡查、查到的數字算不算數。

### 2.1 台灣都市計畫層級體系

```
【第一層】國土計畫
│  法源：國土計畫法（2016年施行）
│  主管：內政部國土管理署
│  確定：土地是哪種「國土功能分區」
│        ├── 城鄉發展地區  → 可以有都市計畫（往下繼續查）
│        ├── 農業發展地區  → 原則不可開發建築
│        ├── 海洋資源地區  → 特殊管制
│        └── 自然保育地區  → 嚴格限制開發
│
▼
【第二層】都市計畫 — 主要計畫
│  法源：都市計畫法
│  主管：縣市政府都市發展局（或建設局）
│  確定：土地屬於哪個「使用分區」
│        ├── 住宅區（住一/住二/住三/住四）
│        ├── 商業區
│        ├── 工業區
│        ├── 行政區、文教區、倉儲區等
│        └── 公共設施用地（道路/公園/學校/停車場等）
│  ⚠️ 主要計畫通常每 25 年通盤檢討一次，
│     但可能有局部變更，查詢時要看「最新版本」
│
▼
【第三層】都市計畫 — 細部計畫
│  法源：都市計畫法第 22 條
│  主管：縣市政府（或內政部核定）
│  確定：各街廓更細緻的管制規定
│        ├── 可能訂定特定街廓的「指定容積率」
│        ├── 建築退縮的特殊要求
│        ├── 開放空間留設義務
│        └── 特殊景觀、文資緩衝區等附加管制
│  ⚠️ 並非所有地區都有細部計畫；
│     有細部計畫的地區，以細部計畫管制為優先
│
▼
【第四層】土地使用分區管制規則
│  法源：都市計畫法第 39 條、各縣市自治條例
│  主管：各縣市政府
│  確定：各使用分區的具體管制數值
│        ├── 建蔽率上限
│        ├── 容積率上限（基準容積）
│        ├── 退縮深度
│        ├── 高度限制
│        └── 准許用途（可以做住宅/商業/哪些行業）
│
▼
【第五層】建築基地層級
   法源：建築法、建築技術規則
   主管：縣市政府建管機關
   確定：個別基地的建築規定
         ├── 指定建築線（建管核發）
         ├── 停車位數量（建技規）
         ├── 無障礙設施（建技規）
         └── 敷地計劃設計 → 建照申請
```

### 2.2 為什麼層級架構這麼重要？

**情境 A：同樣是「住宅區」，不同地方數字就是不一樣**

| 地點 | 使用分區 | 容積率 | 備註 |
|------|---------|--------|------|
| 台北市大安區 | 第三種住宅區 | 225% | 依台北市自治條例 |
| 新北市板橋區某地 | 第三種住宅區 | 160% | 細部計畫另有規定 |
| 台中市某地 | 第三種住宅區 | 300% | 依台中市自治條例 |

> 同樣叫「住三」，但數字不同，原因就是**各縣市的第四層「管制規則」不同**。
> 甚至同一縣市的不同街廓，細部計畫可能還有額外限制。

**情境 B：開發業者常見的錯誤**

```
錯誤做法：
  看到「住三」→ 直接用 225% 試算容積
  → 但細部計畫訂這個街廓只有 160%
  → 規模估算整個錯

正確做法：
  第二層：確認分區（住三）
  第三層：查細部計畫有無特別規定 ← 這步很多人跳過
  第四層：查縣市管制規則的數值
  → 才確認基準容積是多少
```

### 2.3 各層計畫的查詢管道

| 層級 | 文件名稱 | 哪裡查 |
|------|---------|--------|
| 第一層 | 國土計畫（縣市版） | 內政部國土管理署網站 |
| 第二層 | 都市計畫主要計畫書、計畫圖 | 縣市政府都市發展局 / 都計書查詢系統 |
| 第三層 | 都市計畫細部計畫書 | 縣市政府都市發展局（有些須現場索取） |
| 第四層 | 土地使用分區管制規則（自治條例） | 各縣市政府法規查詢系統 |
| 第五層 | 指定建築線 | 縣市建管處申請 |

> 實務捷徑：台灣各縣市都有「都市計畫書圖資訊系統」（GIS），
> 輸入地號就可以查到第二層的使用分區。但**第三層細部計畫務必另外再確認**，
> GIS 系統不一定會顯示細部計畫的特殊管制。

## 3. 從層級架構找到你的管制數值

### 3.1 查詢步驟（新案前期評估標準流程）

```
步驟 1：取得地籍謄本
  → 確認地號、地目、面積
  → 確認所有人（是否有他項權利）

步驟 2：查都市計畫（第二層）
  → 使用都計GIS系統輸入地號
  → 確認：使用分區、公共設施用地比例
  → 確認：有無計畫道路（會影響退縮與地積）

步驟 3：查細部計畫（第三層）
  → 向縣市都發局索取或查詢
  → 確認：有無特定街廓管制
  → 確認：有無開放空間義務、特殊景觀規定

步驟 4：查土地使用分區管制規則（第四層）
  → 取得該縣市最新版自治條例
  → 查表取得：建蔽率上限、容積率（基準容積）
  → 確認：退縮規定、高度限制

步驟 5：申請指定建築線（第五層）
  → 向縣市建管處申請
  → 取得指定建築線位置與寬度
  → 這才是計算退縮的基準

步驟 6：確認容積獎勵（如適用）
  → 都更：確認更新單元內的獎勵容積計算
  → 危老：確認各項獎勵的申請資格
```

### 3.2 容積數值的完整推導路徑

```
基準容積（來自第四層管制規則）
  + 細部計畫特殊容積（第三層，如有）
  + 容積獎勵（都更獎勵 or 危老獎勵）
  + 開放空間容積獎勵（第五層，如留設符合規定）
  ─────────────────────────────────
  = 可建容積（這才是實際可用的上限）

可建容積 × 基地面積 = 可建總樓地板面積（扣除免計後）
```

## 4. 敷地計劃書的必備內容

### 4.1 書面計劃書（文字部分）

**第一節：基地都市計畫概述（對應層級架構）**

| 應說明事項 | 資料來源 |
|----------|---------|
| 國土功能分區 | 縣市國土計畫 |
| 都市計畫名稱及核定日期 | 主要計畫書 |
| 使用分區 | 主要計畫圖 |
| 細部計畫有無特殊管制 | 細部計畫書 |
| 基準容積（建蔽率／容積率） | 土地使用分區管制規則 |
| 計畫道路寬度 | 主要計畫圖 + 指定建築線 |

> 這一節是整份敷地計劃書的「法規依據起點」，
> 把上面的資訊列清楚，後面所有計算才有根據。

**第二節：法規查核摘要**
- 建蔽率計算
- 容積率計算（含獎勵）
- 退縮規定
- 停車位計算
- 無障礙設施規劃

**第三節：配置說明**
- 建物棟數、樓層數
- 各用途樓地板面積彙整
- 法定空地位置與面積
- 開放空間說明（若適用）

**第四節：環境影響（若需要）**
- 日照分析、通風分析、景觀敏感區說明

### 4.2 必備圖面

| 圖號 | 圖面名稱 | 比例 | 重點標示 |
|------|---------|------|---------|
| S-01 | 位置示意圖 | 1/1200 或 1/600 | 基地四至、周邊道路 |
| S-02 | 都計分區套圖 | 1/500 | 使用分區界線、計畫道路 |
| S-03 | 地籍套圖 | 1/500 | 地號、道路寬度、地界 |
| S-04 | 配置圖 | 1/200 或 1/300 | 退縮線、建物外廓、停車 |
| S-05 | 法定空地圖 | 1/200 | 法定空地範圍、面積計算 |
| S-06 | 停車平面圖 | 1/100 | 車位編號、尺寸、迴車動線 |

> S-02「都計分區套圖」在 v1.0 版本遺漏，**這張圖是對應層級架構的關鍵圖面**，
> 要顯示基地位於哪個使用分區、計畫道路在哪、細部計畫特殊管制線在哪。

## 5. 關鍵計算公式與查核要點

### 5.1 建蔽率計算

```
建蔽率 = 建築面積 ÷ 基地面積 × 100%

建築面積 = 各層最大投影面積（不含免計項目）

免計入建蔽率項目（常見）：
  - 地下層（含地下停車場）
  - 陽台（符合規定尺寸內）
  - 法定騎樓
```

**查核重點：**
- 建築面積是否超過第四層管制規則的上限
- 屋簷挑出超過 1.5m 以上部分是否已計入

### 5.2 容積率計算

```
容積率 = 各層樓地板面積總和 ÷ 基地面積 × 100%

樓地板面積總和 = Σ（各層實際樓地板面積）
              - 免計容積項目

常見免計容積項目：
  - 停車空間（附建停車位）
  - 機電設備空間（限定比率）
  - 屋頂突出物（電梯機房、樓梯間）
  - 開放空間獎勵（若適用）
```

**查核重點：**
- 基準容積來源是否已對應第三、四層管制規則
- 免計項目是否有超過法定比率上限
- 地下一層部分用途是否需計入容積

### 5.3 退縮計算

```
退縮建築線 = 指定建築線（第五層，建管核發）+ 附加退縮

附加退縮規定（依各縣市第四層管制規則，以台北市住宅區為例）：
  - 建築物高度 ≤ 15m：退縮 2m
  - 建築物高度 15m < H ≤ 21m：退縮 3m
  - 建築物高度 > 21m：退縮 4m
```

> ⚠️ **特別提醒**：計畫道路寬度（第二層主要計畫訂定）
> 與現況道路寬度可能不同。退縮一定要從**指定建築線**算起，
> 不是從現況路面邊緣算起。

### 5.4 停車位計算

```
應設停車位數 = 住宅樓地板面積 ÷ 每輛所需面積標準

台北市範例（住宅）：
  - 每 75㎡ 樓地板面積設 1 輛（基本）
  - 機械停車位換算：1 機械位 = 0.7 輛（依規）

停車空間面積（每輛）：
  - 小型車：長 5.5m × 寬 2.5m
  - 機械停車：依廠商規格另計
```

## 6. 都更／危老特別注意事項

### 6.1 層級架構在都更中的特殊意義

都更案件有額外的「事業計畫書」，等同在第四層之上**再疊加一層**：

```
【都更事業計畫】（特殊層）
│  法源：都市更新條例
│  確定：更新單元範圍、容積獎勵、公共設施比例
│  ⚠️ 都更事業計畫核定後，
│     才算確定可建容積；
│     計畫核定前的試算只是估算
│
▼
【敷地計劃】（依都更事業計畫設計）
```

### 6.2 危老重建敷地計劃重點

**基地條件確認：**
```
危老申請資格查核清單
□ 建物屋齡 ≥ 30 年（民國○年以前完工）
□ 結構安全評估（耐震能力不足 or 海砂屋/輻射屋）
□ 基地面積 ≥ 200㎡（或符合例外規定）
□ 全體所有權人同意（或 10/10）
□ 所在地使用分區允許住宅（查第四層管制規則）
```

**容積獎勵計劃書附件：**
- 結構安全性能評估報告（取得獎勵必備）
- 綠建築設計說明（若申請綠建築獎勵）
- 智慧建築設計說明（若申請智慧建築獎勵）

### 6.3 都市更新敷地計劃重點

**事業概要 vs 事業計劃：**
```
事業概要階段（較早）：
  └── 概略敷地計劃即可（位置、規模、容積試算）
      ⚠️ 此時的容積試算要標明「含預估獎勵」，尚未核定

事業計劃階段（送審）：
  └── 完整敷地計劃書（含詳細計算書、圖面）
      ⚠️ 此時容積已依都更計畫核定，數字才是確定值
```

**更新後敷地計劃書特殊內容：**
- 更新前後對照表（現況 vs 計劃）
- 公共設施用地比例說明
- 開放空間留設計劃（影響容積獎勵）

### 6.4 整合型基地的敷地計劃挑戰

| 問題 | 說明 | 對策 |
|------|------|------|
| 地號整合 | 多筆地號合併計算容積 | 先辦地籍整理或合件申請 |
| 分區不同 | 基地跨不同使用分區 | 各分區容積分開計算後加總 |
| 細部計畫不一致 | 基地跨兩個細部計畫範圍 | 分別查明，不可直接平均 |
| 道路退縮不一致 | 不同側道路寬度不同 | 逐條邊界分別計算退縮 |
| 既有建物保留 | 都更不拆部分建物 | 分棟計算容積，再合併確認 |

## 7. 常見審查缺失與對策

### 7.1 層級查詢類（根本性錯誤）

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 容積率用錯 | 只查了縣市通則，漏查細部計畫 | 每案必須確認第三層細部計畫有無特別規定 |
| 計畫道路退縮不足 | 以現況路寬計算，未查主要計畫 | 向建管申請指定建築線並附計畫道路寬度確認文件 |
| 使用分區認定錯誤 | GIS 查詢誤讀分區界線 | 以都市計畫主要計畫圖正本為準，請都發局確認 |

### 7.2 建蔽率 / 容積率類

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 建蔽率超過上限 | 陽台外突計算有誤 | 重新量算各層最大投影面積 |
| 容積率超過上限 | 免計項目誤計 | 逐項確認免計依據法條 |
| 計算基礎有誤 | 用公告地積而非實測地積 | 以地政事務所實測成果為準 |

### 7.3 退縮 / 配置類

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 退縮深度不足 | 漏算道路拓寬計劃 | 查詢主要計畫預定道路計劃（第二層） |
| 建物跨越建築線 | AutoCAD 圖面精度問題 | 以坐標套算，精度至 cm |
| 停車迴車空間不足 | 機械停車規劃過密 | 依機械停車廠商圖面配置 |

### 7.4 法定空地類

| 缺失類型 | 常見原因 | 改善對策 |
|---------|---------|---------|
| 法定空地不足 | 建蔽率算到上限時空地剛好不足 | 反推：先計算法定空地最小面積再配置建物 |
| 法定空地被覆蓋 | 地下室頂板未計入透水面積 | 確認綠化及透水規定，分開計算 |

## 8. 自我檢查清單（送件前）

### 8.1 都市計畫層級查核（最優先確認）

```
□ 已查國土計畫 → 確認城鄉發展地區
□ 已查主要計畫書圖 → 確認使用分區（有文件存檔）
□ 已查細部計畫 → 確認有無特殊管制（有文件存檔）
□ 已查土地使用分區管制規則 → 取得建蔽率、容積率數值
□ 已取得指定建築線 → 知道退縮從哪條線算起
```

### 8.2 基本資料確認

```
□ 地籍謄本（近3個月）
□ 地籍圖謄本（確認地界）
□ 都計書及計畫道路圖
□ 道路現況照片（四至方向各一張）
```

### 8.3 計算書查核

```
□ 建蔽率計算書（附圖面標示）
□ 容積率計算書（逐層列表，含獎勵來源說明）
□ 各層免計項目計算（有法條依據）
□ 停車位計算書（住宅、商業分開）
□ 無障礙設施數量確認
□ 法定空地面積計算書
```

### 8.4 圖面一致性

```
□ S-02 都計分區套圖已附（新增必備圖面）
□ 配置圖退縮線標示正確（從指定建築線起算）
□ 各圖面比例標示一致
□ 北向箭頭方向正確
□ 樓層標示（FL、GL）定義一致
□ 建物外廓尺寸與計算書一致
```

### 8.5 都更／危老特別查核

```
□ 容積獎勵計算有文件依據（且已對應細部計畫規定）
□ 獎勵申請書類與計劃書數字一致
□ 原建物屋齡證明文件備齊
□ 都更事業計畫（或危老申請書）與敷地計劃一致
```

## 9. 名詞解釋速查

| 名詞 | 英文 | 說明 |
|------|------|------|
| 國土計畫 | National Land Plan | 最上位的空間計畫，確定土地功能分區 |
| 主要計畫 | Master Plan | 都市計畫的上位計畫，確定使用分區與公設用地 |
| 細部計畫 | Detailed Plan | 依主要計畫擬定的細節管制，優先適用 |
| 土地使用分區管制規則 | Zoning Regulations | 各縣市訂定各分區管制數值的自治規範 |
| 敷地計劃 | Site Plan | 基地整體配置計劃書 |
| 基地面積 | Site Area | 計算建蔽率、容積率的基準面積 |
| 建蔽率 | Building Coverage Ratio (BCR) | 建築面積÷基地面積 |
| 容積率 | Floor Area Ratio (FAR) | 樓地板面積總和÷基地面積 |
| 建築面積 | Building Footprint | 建物各層最大水平投影面積 |
| 法定空地 | Required Open Space | 建蔽率管制下的未建築地 |
| 退縮 | Setback | 建物退離指定建築線的距離 |
| 建築線 | Building Line | 建管指定的建築許可起造線 |
| 指定建築線 | Designated Building Line | 建管單位依計畫道路指定的線（退縮起算點）|
| 計畫道路 | Planned Road | 主要計畫訂定的預定道路（可能比現況寬）|
| 容積獎勵 | FAR Bonus | 符合條件可額外增加的容積 |
| 基準容積 | Base FAR | 土地使用分區管制規則訂定的容積上限 |
| 可建容積 | Allowable FAR | 含獎勵後實際可使用的容積 |
| 樓地板面積 | Floor Area | 各層樓板面積（含牆體） |
| 免計容積 | FAR Exemption | 不納入容積計算的面積項目 |
| 附建停車位 | Required Parking | 法定應附設的停車空間 |
| 開放空間 | Open Space | 供公眾使用的留設空地（可換取容積獎勵）|
| 無障礙設施 | Barrier-free Facilities | 供行動不便者使用的設備設施 |
