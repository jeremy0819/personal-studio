## 1. 名詞定義

### Q：何謂「敷地計劃」？

**答：**
敷地計劃（Site Plan）係指建築申請時，就基地範圍內之建築物配置、退縮、法定空地、停車、景觀及無障礙設施等空間關係，以計劃書與圖說表現之整體配置計劃。

依**建築法**規定，申請建築許可應附具敷地計劃書，作為建管機關審查之依據。

---

### Q：敷地計劃與都市計畫的關係為何？

**答：**
敷地計劃為「個別基地層級」之計劃，須符合上位都市計畫所訂定之各項管制規定。其關係如下：

```
上位計畫（都市計畫各層級）
        ↓ 訂定管制數值
敷地計劃（個別基地配置）
        ↓ 依規申請
建照核發
```

敷地計劃不得牴觸上位計畫，若有衝突，以上位計畫規定為準。

## 2. 都市計畫層級體系

### Q：試述台灣都市計畫之層級體系。

**答：**
台灣都市計畫依效力高低，分為以下五個層級：

---

**【第一層】國土計畫**

| 項目 | 內容 |
|------|------|
| 法源 | 國土計畫法（2016 年施行） |
| 主管機關 | 內政部國土管理署 |
| 功能 | 劃定四種「國土功能分區」 |

四種國土功能分區：

| 分區 | 建築申請 |
|------|---------|
| 城鄉發展地區 | 可適用都市計畫 → 往下查 |
| 農業發展地區 | 原則不得申請一般建築 |
| 海洋資源地區 | 特殊管制 |
| 自然保育地區 | 嚴格限制開發 |

---

**【第二層】都市計畫主要計畫**

| 項目 | 內容 |
|------|------|
| 法源 | 都市計畫法 |
| 主管機關 | 縣市政府都市發展局 |
| 功能 | 劃定使用分區、公共設施用地 |
| 通盤檢討 | 每 25 年一次（可辦理局部變更） |

常見使用分區：住宅區（住一至住四）、商業區、工業區、行政區、文教區、公共設施用地。

> ⚠️ **考試重點**：主要計畫訂定「計畫道路寬度」，與現況路寬可能不同，退縮計算以計畫道路寬度為準。

---

**【第三層】都市計畫細部計畫**

| 項目 | 內容 |
|------|------|
| 法源 | 都市計畫法第 22 條 |
| 主管機關 | 縣市政府（或報內政部核定） |
| 功能 | 訂定各街廓詳細管制規定 |

細部計畫得另訂：
- 特定街廓之指定容積率（可低於通則規定）
- 附加退縮規定
- 開放空間留設義務
- 特殊景觀保護、文化資產緩衝區管制

> ⚠️ **考試重點**：有細部計畫之地區，**以細部計畫規定優先適用**，不得逕以縣市通則規定計算。

---

**【第四層】土地使用分區管制規則**

| 項目 | 內容 |
|------|------|
| 法源 | 都市計畫法第 39 條、各縣市自治條例 |
| 主管機關 | 各縣市政府 |
| 功能 | 訂定各分區之建蔽率、容積率、退縮、高度及准許用途 |

常見住宅區管制數值（通例，各縣市有差異）：

| 分區 | 建蔽率 | 容積率（基準） |
|------|--------|--------------|
| 第一種住宅區 | 30% | 60% |
| 第二種住宅區 | 35% | 120% |
| 第三種住宅區 | 45% | 225% |
| 第四種住宅區 | 50% | 300% |
| 商業區 | 70% | 依縣市差異大 |

---

**【第五層】建築基地層級**

| 項目 | 內容 |
|------|------|
| 法源 | 建築法、建築技術規則 |
| 主管機關 | 縣市建管機關 |
| 核發文件 | 指定建築線 |
| 管制內容 | 停車位數量、無障礙設施、敷地計劃設計 |

**指定建築線**：建管機關依計畫道路所核發之建築許可起造線，為計算退縮之基準。

---

### Q：各層計畫之查詢管道為何？

**答：**

| 層級 | 文件名稱 | 查詢管道 |
|------|---------|---------|
| 第一層 | 縣市國土計畫 | 內政部國土管理署網站 |
| 第二層 | 都市計畫主要計畫書圖 | 縣市都市發展局 / 都計書GIS系統 |
| 第三層 | 都市計畫細部計畫書 | 縣市都市發展局（部分需現場索取） |
| 第四層 | 土地使用分區管制規則 | 各縣市政府法規查詢系統 |
| 第五層 | 指定建築線 | 縣市建管處申請 |

## 3. 敷地計劃書的組成

### Q：敷地計劃書應包含哪些內容？

**答：**
敷地計劃書通常分為文字計劃書與圖面兩部分。

**（一）文字計劃書**

1. **基地都市計畫概述**
   - 國土功能分區、都市計畫名稱及核定日期
   - 使用分區、細部計畫有無特殊管制
   - 基準容積（建蔽率、容積率）
   - 計畫道路寬度

2. **法規查核摘要**
   - 建蔽率、容積率計算
   - 退縮規定、停車位計算
   - 無障礙設施規劃

3. **配置說明**
   - 建物棟數、樓層數
   - 各用途樓地板面積彙整
   - 法定空地位置與面積
   - 開放空間留設（若適用）

**（二）必備圖面**

| 圖號 | 圖面名稱 | 常用比例 | 應標示重點 |
|------|---------|---------|-----------|
| S-01 | 位置示意圖 | 1/1200 或 1/600 | 基地四至、周邊道路 |
| S-02 | 都計分區套圖 | 1/500 | 使用分區界線、計畫道路位置 |
| S-03 | 地籍套圖 | 1/500 | 地號、道路寬度、地界 |
| S-04 | 配置圖 | 1/200 或 1/300 | 退縮線、建物外廓、停車 |
| S-05 | 法定空地圖 | 1/200 | 法定空地範圍、面積計算 |
| S-06 | 停車平面圖 | 1/100 | 車位編號、尺寸、迴車動線 |

## 4. 建蔽率與容積率計算

### Q：建蔽率之定義及計算方式為何？

**答：**

**定義**：建蔽率係建築面積占基地面積之比率，用以管制基地內建築物之水平覆蓋程度，確保法定空地之留設。

**計算公式**：
```
建蔽率（%）= 建築面積 ÷ 基地面積 × 100%
```

**建築面積**：建築物各層最大投影面積，不含下列免計項目：

| 免計項目 | 條件 |
|---------|------|
| 地下層 | 含地下停車場 |
| 陽台 | 符合法定尺寸（建技規第 164 條） |
| 法定騎樓 | 供公眾通行之騎樓 |
| 雨遮、屋簷 | 挑出未超過 1.5m 部分 |

> ⚠️ **注意**：屋簷或雨遮挑出**超過 1.5m** 之部分，應計入建築面積。

---

### Q：容積率之定義及計算方式為何？

**答：**

**定義**：容積率係建築物各層樓地板面積總和（容積總樓地板面積）占基地面積之比率，用以管制基地內之開發強度。

**計算公式**：
```
容積率（%）= 容積總樓地板面積 ÷ 基地面積 × 100%

容積總樓地板面積 = 各層樓地板面積總和 − 免計容積面積
```

**常見免計容積項目**（依建築技術規則）：

| 免計項目 | 說明 |
|---------|------|
| 附建停車空間 | 法定應設之停車位及其必要通道 |
| 機電設備空間 | 不超過法定比率上限 |
| 屋頂突出物 | 電梯機房、樓梯間等，面積需符合規定 |
| 開放空間獎勵楓積 | 依開放空間設計規定取得 |

> ⚠️ **考試重點**：各免計項目均有**面積上限或比率限制**，不得無限免計。

---

### Q：可建容積之計算方式為何？

**答：**
```
可建容積 = 基準容積（第四層管制規則）
         + 細部計畫特別指定容積（第三層，如有）
         + 容積獎勵（依個別法令申請）
         + 開放空間容積獎勵（如留設符合規定）

可建總樓地板面積 = 可建容積率 × 基地面積
```

## 5. 退縮規定

### Q：何謂建築退縮？指定建築線與退縮有何關係？

**答：**

**建築退縮**係指建築物外牆面退離指定建築線之距離，用以留設人行空間、確保道路視線及防火間距。

**指定建築線**：建管機關依計畫道路所核定之建築起造線，為退縮計算之**起算基準**。

```
退縮後建築線 = 指定建築線 + 附加退縮深度
```

> ⚠️ **考試易錯點**：退縮應自「指定建築線」起算，而非自**現況路面邊緣**起算。
> 計畫道路寬度若大於現況道路，指定建築線會向內退，導致可建地積減少。

---

### Q：台北市住宅區之退縮規定為何？（以臺北市為例）

**答：**

依臺北市土地使用分區管制自治條例（附加退縮規定）：

| 建築物高度 | 應退縮深度 |
|----------|-----------|
| H ≦ 15m | 2m |
| 15m < H ≦ 21m | 3m |
| H > 21m | 4m |

注意：各縣市規定不同，考試應以**題目指定縣市法規**為準。

## 6. 停車位附設規定

### Q：建築物應附設停車空間之計算方式為何？

**答：**

依建築技術規則建築設計施工編第 59 條及各縣市規定，應依**建築物用途及樓地板面積**計算應設停車位數。

**計算公式**（以台北市住宅為例）：
```
應設停車位數 = 住宅樓地板面積 ÷ 每輛基準面積（75㎡/輛）
```

**停車位最小尺寸**（建技規第 62 條）：

| 車種 | 長度 | 寬度 |
|------|------|------|
| 小型車 | 5.5m | 2.5m |
| 機車 | 2.0m | 1.0m |

**機械停車位換算**：
```
機械停車位 × 0.7 = 折算停車位數（各縣市換算率不同）
```

> ⚠️ **考試重點**：停車空間（及其通道）屬**免計容積**項目，但不得超過法定比率上限。

## 7. 法定空地

### Q：何謂法定空地？其與建蔽率有何關係？

**答：**

**法定空地**係因建蔽率管制，基地內未被建築物覆蓋之土地。

```
法定空地面積 = 基地面積 × （1 − 建蔽率上限）

最小法定空地面積 ≥ 基地面積 × （1 − 建蔽率上限）
```

**法定空地之規定**（建築技術規則第 24 條）：
- 法定空地應保留空曠，供通風採光之用
- 不得舖設妨礙排水之不透水鋪面（有透水率規定之縣市）
- 地下室頂板上方之法定空地，仍須計算透水及綠化面積

> ⚠️ **常見錯誤**：建蔽率用足上限後，若忘記驗算法定空地面積是否符合透水率規定，會導致補件。

---

### Q：法定空地與開放空間有何區別？

**答：**

| 項目 | 法定空地 | 開放空間 |
|------|---------|---------|
| 性質 | 消極留設（不得建築） | 積極提供（供公眾使用） |
| 法源 | 建蔽率管制（建技規） | 獎勵規定（都計法或地方規定） |
| 效果 | 無容積獎勵 | 可換取容積獎勵 |
| 管理 | 基地所有權人 | 公眾使用，地主仍擁有所有權 |

## 8. 歷屆重點考題解析

### 題型一：建蔽率 / 容積率計算

**範例題**

> 某基地面積 500㎡，位於第三種住宅區（建蔽率 45%、容積率 225%）。
> 建築物地上 10 層，各層樓地板面積均為 180㎡，附建停車位 60㎡（免計容積）。
> 試計算：（1）建蔽率是否符合規定？（2）容積率是否符合規定？

**解題步驟**：

（1）建蔽率驗算：
```
建築面積 = 180㎡（各層最大投影面積，取 1F）
建蔽率 = 180 ÷ 500 × 100% = 36%
36% ≦ 45% → 符合規定 ✓
```

（2）容積率驗算：
```
各層樓地板面積總和 = 180㎡ × 10 層 = 1,800㎡
免計容積（停車）= 60㎡
容積總樓地板面積 = 1,800 − 60 = 1,740㎡
容積率 = 1,740 ÷ 500 × 100% = 348%
348% > 225% → 超過規定上限 ✗
```

**結論**：建蔽率符合，容積率超過，設計需縮減樓地板面積。

---

### 題型二：退縮計算

**範例題**

> 基地臨 12m 計畫道路，現況道路寬度為 8m，道路中心線位置不變。
> 依規定建物應退縮 2m（自指定建築線起算）。
> 試問：建築物外牆面距現況路面邊緣應有多少距離？

**解題步驟**：
```
計畫道路半寬 = 12m ÷ 2 = 6m（自道路中心線）
現況道路半寬 = 8m ÷ 2 = 4m（自道路中心線）

指定建築線距道路中心線 = 6m
現況路面邊緣距道路中心線 = 4m

指定建築線距現況路面邊緣 = 6 − 4 = 2m（需讓地）

建物外牆距指定建築線 = 退縮 2m

建物外牆距現況路面邊緣 = 2（讓地）+ 2（退縮）= 4m
```

---

### 題型三：都市計畫層級辨析

**範例題**

> 甲縣市土地使用分區管制規則規定第三種住宅區容積率為 225%，
> 但該基地所在街廓之細部計畫規定容積率為 160%。
> 問：該基地之基準容積應以何者為準？

**答**：

依都市計畫法第 22 條，細部計畫為依主要計畫所擬定之詳細計畫，具有更高之特定性。
**有細部計畫之地區，以細部計畫規定優先適用。**

因此，該基地基準容積應以細部計畫規定之 **160%** 為準，不得適用縣市通則之 225%。

---

### 題型四：法定空地計算

**範例題**

> 基地面積 400㎡，第二種住宅區（建蔽率 35%）。
> 問：法定空地最小面積為何？

**解題**：
```
法定空地最小面積 = 基地面積 × （1 − 建蔽率上限）
                = 400 × （1 − 35%）
                = 400 × 65%
                = 260㎡
```

## 9. 名詞速查表

| 名詞 | 定義 |
|------|------|
| 國土計畫 | 最上位空間計畫，確定土地國土功能分區 |
| 主要計畫 | 都市計畫上位計畫，劃定使用分區與公共設施用地 |
| 細部計畫 | 依主要計畫擬定，訂定各街廓詳細管制，優先適用 |
| 土地使用分區管制規則 | 各縣市自治條例，訂定各分區管制數值 |
| 敷地計劃 | 建照申請時之基地整體配置計劃書 |
| 指定建築線 | 建管依計畫道路所核發之建築起造線（退縮起算點）|
| 計畫道路 | 主要計畫訂定之預定道路寬度（可能大於現況路寬）|
| 建蔽率 | 建築面積 ÷ 基地面積，管制基地水平覆蓋程度 |
| 容積率 | 容積樓地板面積 ÷ 基地面積，管制基地開發強度 |
| 建築面積 | 各層最大水平投影面積（扣除免計項目） |
| 容積總樓地板面積 | 各層樓地板面積總和扣除免計容積後之面積 |
| 法定空地 | 因建蔽率管制，基地內未被建物覆蓋之空地 |
| 基準容積 | 分區管制規則訂定之容積率上限 |
| 可建容積 | 基準容積加計各項容積獎勵後之實際上限 |
| 免計建蔽率面積 | 不計入建蔽率計算之部分（陽台、地下層等）|
| 免計容積面積 | 不計入容積計算之部分（停車、機電設備等）|
| 開放空間 | 留設供公眾使用之空地，可申請容積獎勵 |
| 無障礙設施 | 依建技規設置，供行動不便者使用之設備設施 |
