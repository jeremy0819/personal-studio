# personal-studio — 我的 AI 工作環境

我的 Claude Code 個人工作室。把 AI skills/agent 當成不同部門的員工管理，
目標是成為建築業的「AI 落地接線人」。

## 內容

| 檔案 | 作用 |
|---|---|
| `CLAUDE.md` | 工作室憲法 / 跨 session 記憶。**每次開新對話請先讀這裡。** |
| `WORKFLOW.md` | 日/週/月迴圈 + 建造順序 + 召喚速查 |
| `.claude/skills/personal-studio/SKILL.md` | 調度大腦（分流派工） |
| `.claude/agents/` | 已招募員工：`mentor`、`auditor`、`evaluation-analyst`、`document-specialist` |

## 怎麼用

在這個 repo 開一個 Claude Code session，說一句「**啟動工作室**」或直接丟任務，
調度 skill 會判斷該派哪個部門員工，或留在對話陪你做。

> 詳見 `CLAUDE.md`。
